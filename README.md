# ♻️ 垃圾分類識別器

一個完全在前端執行的垃圾分類 AI 應用，支援實時攝像頭識別和圖片上傳分類。

## 🎯 功能特性

- **📷 實時攝像頭識別** - 使用網頁攝像頭進行即時垃圾分類
- **📁 圖片上傳** - 上傳圖片進行分類
- **🧠 AI 模型** - 基於 TensorFlow.js 的 MobileNet 模型
- **🔒 完全隱私** - 所有處理都在本地浏覽器進行，無需上傳數據
- **📱 響應式設計** - 支援桌用和行動設備
- **🌍 多語言** - 繁體中文界面

## 🏷️ 分類種類

應用可以識別以下三種垃圾類型：

1. **🔵 玻璃 (Glass)** - 玻璃瓶、玻璃杯、玻璃罐等
2. **🟫 垃圾 (Trash)** - 一般廢棄物
3. **📄 紙張 (Paper)** - 紙盒、紙張、紙板等

## 🚀 如何使用

### 方法 1：攝像頭識別

1. 打開應用
2. 點擊「📷 攝像頭」標籤
3. 點擊「🎥 啟動攝像頭」按鈕
4. 將垃圾對準攝像頭
5. 應用會自動識別並顯示分類結果

### 方法 2：上傳圖片

1. 打開應用
2. 點擊「📁 上傳圖片」標籤
3. 拖拽圖片到上傳區域或點擊上傳
4. 點擊「🔍 開始分類」按鈕
5. 查看分類結果

## 💡 使用技巧

1. **光線充足** - 確保周圍有足夠光線以獲得最佳識別效果
2. **清晰圖像** - 物體應清晰可見於鏡頭
3. **單個物體** - 一次識別單個垃圾物體效果最好
4. **不同角度** - 嘗試從不同角度拍攝以提高準確度
5. **背景簡潔** - 簡單的背景能提供更好的識別效果

## 🔐 隱私和安全

- ✅ **完全離線處理** - 所有圖像處理都在您的本地浏覽器上進行
- ✅ **無數據上傳** - 沒有任何數據被發送到遠程服務器
- ✅ **本地存儲** - 攝像頭數據不被保存
- ✅ **開源透明** - 所有代碼都是可公開查看的

## 📊 模型性能

- **模型**: MobileNet V2
- **輸入大小**: 224x224 像素
- **類別數**: 3（玻璃、垃圾、紙張）
- **精度**: ~85-90%（取決於圖像質量）
- **推斷時間**: ~500-1000ms（取決於設備）
